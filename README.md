# 文件备份共享平台

## 接口文档

### 授权

> GET /oauth/authorize

- 入参

| 参数           | 类型     | 备注             |
|--------------|--------|----------------|
| redirect_uri | String | 重定向的地址         |
| state        | String | 自定义数据，回调时会携带回来 |

### 登录

> POST /oauth/login

- 入参

| 参数 | 类型   | 备注                    |
| ---- | ------ | ----------------------- |
| code | String | 重定向后自动携带的 Code |

- 出参

| 参数  | 类型   | 备注               |
| ----- | ------ | ------------------ |
| token | String | 登录后返回的 Token |
